Flutter 使用 iconfont 字体

项目中难免我们会用到各种各样的字体,在Flutter中可能字体会存在一些局限性, 所以我们可以引入三方字体在Flutter中使用

第一步

打开字体网站 https://www.iconfont.cn/
选择所需要的图标点击下载到本地

第二步

打开项目/目录下创建static/fonts文件夹
将下载好的iconfont.ttf复制到static/fonts文件夹中

第三步

配置pubspec.yaml文件如下图
配置完成后就可以在项目中使用了

使用方法

查看iconfont.css，将\替换成0x即可得到对应的 codePoint

.icon-ai-ios:before {
  content: "\e631";
}

IconData(0xe631, fontFamily: 'iconfont')
